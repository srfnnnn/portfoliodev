Overview

Task list manager helps users organise and manage their daily activities efficiently. Users can add, edit, delete and track tasks, as well as view their overall completion status. Tasks can be categorised under predefined categories such as grocery shopping, errands, schoolwork and household chores, making it easier to organise and identify responsibilities.


Users can add new tasks, assign them to specific categories, update task details, mark tasks as completed or delete tasks when they are no longer needed. An overall status feature provides a quick summary of task progress, helping users stay motivated and on track.
When adding a task, users enter a description and select a category. Tasks can be edited to update the description, change the category, or mark them as completed using a toggle switch. A confirmation pop-up is shown before deleting tasks to prevent accidental removal. Input validation using the .trim() method prevents empty tasks from being added and each task is assigned a unique ID to ensure accurate editing and deletion.


Key Features

Task Categorisation


  • Tasks can be grouped under different categories
  • Helps users organise tasks clearly and avoid clutter

Add Tasks


  • Users can create new tasks and assign them to a selected category
  • Makes task planning simple and structured

Edit Tasks


  • Existing tasks can be edited at any time
  • Users can update task names, categories and completion status
  • Each task can be marked as Completed or Not Completed
  • Allows users to easily track progress at a glance

Delete Tasks


  • Tasks can be removed when they are no longer needed
  • Keeps the task list clean and relevant

Overall Status Tracking


  • Displays the total number of completed and incomplete tasks
  • Shows the percentage of completed tasks to give users a clear overview of their productivity


Purpose

The app is designed to help users to stay organised, manage time more effectively, reduce confusion from unstructured task lists and visually track progress and completion
