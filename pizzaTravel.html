Web Ordering Application

Fah’s Pizza Co. is an interactive pizza ordering web application designed to provide users with a smooth and intuitive online ordering experience. The website showcases a curated menu of pizzas, desserts and sides, allowing users to browse items, add them to a cart and manage their order in real time.


Key Features

Real-time store status


  • The website displays “We’re currently open” based on predefined operating hours set in the code
  • feature dynamically reflects whether the store is open or closed, improving realism and user experience

Live Cart Item Counter


  • A number beside the cart icon shows how many items have been added.
  • This updates instantly as items are added, giving users clear feedback on their order.

Add to Cart with Quantity Control


  • Clicking the “+” button on a menu item reveals a quantity input, allowing users to select how many items they want before adding them to the cart
  • This makes the ordering process more flexible and user-friendly

Dynamic Shopping Cart


  • The cart updates in real time to display — Item name, Quantity and Price
  • Users can clearly review what they’ve added before proceeding

Interactive Menu Display


  • Menu items are visually presented with high-quality images, item names, descriptions and prices
  • Categories such as Top Flavours and Desserts & Sides help users easily navigate the menu


Purpose

This project demonstrates the use of interactive UI elements, state management and conditional rendering to simulate a real-world food ordering system. It highlights practical frontend development skills while focusing on usability and visual design.

Overview

My Travel App is a personal travel planning tool that helps users track trip details and organize their packing lists. The app focuses on trip management rather than booking or destination exploration. It’s simple, user-friendly and visually organized with sections for travel dates, countdowns and packing items.


Key Features


Travel Details Section


  • Destination input: Users can type the location they’re traveling to
  • Trip dates: Users can select start (From) and end (To) dates using a date picker.
  • Trip duration: Automatically calculates and displays the number of days for the trip.
  • Countdown: Shows the days left until the trip, providing a fun visual countdown.

Packing List Section


  • Category selection: Users can choose the type of item from a dropdown
  • Quantity input: Allows users to set how many of each item they need
  • Item input: Users can type the item name and add it to the list
  • Add button: Adds the item to the packing list

Item display

  • Shows items grouped by category
  • Each item has a checkbox to mark as packed
  • Trash icon to delete items
  • Quantity indicated in brackets

Packing summary

  • Shows the total items in the list
  • Displays how many items are already packed, both as a number and percentage


Reset button


It lets users start fresh by clearing their travel list once they’ve completed all planned activities. After marking all items as done, users can easily reset the list, removing all entries so they can plan their next trip from scratch without any leftover tasks. It keeps the app organized and ensures every new journey starts with a clean slate

Purpose

This app is designed to help travelers plan their trips effectively, keeping both dates and packing organized in a single interface. It’s especially useful for short trips where managing essentials quickly is important.
